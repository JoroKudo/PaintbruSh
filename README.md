# PaintbruSh
![Imgur](https://i.imgur.com/6QdhCJy.png)

A terminal based drawing app

##The controls are:
- H move left
- J move down
- L move right
- K move up
- Q quit
- SPACE Lift or lower pen
- [1-7] colors to pick
- S Export Drawing to png

Image export only works if you have [ansilove](https://github.com/ansilove/ansilove) installed
